# 오프닝 인트로 화면 — 후보 모음

타이틀 다음에 나올 **오프닝 인트로**(검은 화면 → 박사 등장 → 텍스트) 구성요소별 후보.
각 폴더에서 맘에 드는 걸 **하나씩** 골라줘. (예: "배경은 dark_03, 박사는 introOak, 포켓몬은 sprigatito")
