## ⚠️ 불꽃(FIRE) 가구 문제
**AR 타일셋 4개를 다 뒤졌지만 "불이 타는 벽난로"가 없다.**
- `All Inner Extended.png`의 벽난로 = 불이 꺼진 회색 벽 아치 (벽 배경이 붙어 나옴)
- `Indoor.PNG`의 불 = 체육관 용암벽/동굴 — 가구가 아님
→ 그래서 현재 불꽃 후보는 **08_램프**뿐. 파이리(FIRE)를 위한 가구가 사실상 비어 있다.

## 규격 (고를 때 참고)
- 한 칸 = 32px. 파일 크기가 곧 칸 수 (예: 64×64 = 2×2칸).
- 가구를 놓은 칸은 **걸어서 지나갈 수 없다**(벽 취급). 방이 좁아지니 큰 가구는 신중히.
- 내 방(2F)에서 **F키 → 꾸미기 모드**로 놓고, **침대 앞 Space → 잠자기**로 컨디션이 오른다.
