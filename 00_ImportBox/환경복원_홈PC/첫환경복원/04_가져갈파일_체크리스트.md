# 04. 가져갈 파일 체크리스트 (git로 오는 것 / 수동으로 챙길 것)

> 집 PC로 옮길 때 "뭐가 자동으로 따라오고, 뭘 손으로 챙겨야 하나" 정리.
> 기준: 프로젝트를 **git clone(또는 폴더 동기화)** 한다고 가정.

---

## ✅ A. git으로 자동으로 따라오는 것 (clone하면 끝)

- `myPokemon_AJ/src/**` (전체 소스), `AGENTS.md`, `CLAUDE.md`, `package.json`, `package-lock.json`, `vite.config.ts`, `tsconfig.json`, `index.html`, `게임실행.bat`
- `myPokemon_AJ/electron/main.cjs`
- `myPokemon_AJ/tools/**` — `fetch-pokemon.mjs`, `import-from-anotherred.mjs`, **`ar-map/extract.py`, `ar-map/rxrender.py`** (단, 스크립트 안 경로는 집 PC에 맞게 수정 필요)
- `myPokemon_AJ/public/assets/**` (~80MB) — fonts, pokemon/front, tilesets, characters, house(추출본+rooms.json), title, intro 등
- `00_ImportBox/**` (이 키트 포함), `01_Resources/**` 중 git에 올린 것

> ⚠️ `public/assets`가 80MB라 git에 잘 올라가 있는지 집에서 `git status`로 한 번 확인. 누락 에셋은 `pokemon-asset-pipeline` skill 절차로 다시 받을 수 있음.

---

## 🔧 B. git에 **없어서** 수동으로 챙겨야 하는 것 (.gitignore됨)

이 폴더 `복사본/`에 다 모아놨다. 그대로 복사해 쓰면 됨.

| 항목 | 어디서 → 어디로 | 비고 |
|---|---|---|
| **Claude skills** (8개) | `복사본/claude_skills/` → `<루트>/.claude/skills/` | `.claude/`가 gitignore라 clone에 없음. (전문은 `00_ImportBox/skills/myPokemon_환경복원_skills.md`에도 박혀 있음) |
| **Claude 메모리** | `복사본/claude_memory/*.md` → 실제 메모리 폴더 | **슬러그 주의!** → `02_게임_핵심설정_메모리.md` §C |
| **settings.local.json** | `복사본/claude_settings/settings.local.json` → `<루트>/.claude/settings.local.json` | Bash 권한 허용목록. 없어도 동작은 함(매번 권한 물어봄) |

## 🌐 C. git 무관 — 별도로 깔거나 받아야 하는 것

| 항목 | 방법 | 비고 |
|---|---|---|
| **어나더레드 원본** `Pokemon Another Red_PWT_250829` | D드라이브에 폴더째 복사(USB/클라우드) | ~수백MB, 에셋·맵의 원천. `D:\Pokemon Another Red_PWT_250829` |
| **node_modules** | `cd myPokemon_AJ && npm install` | Electron 504 시 미러(→ 01 문서) |
| **exe (`build_win/`)** | `npm run build && npx electron-builder --win --x64`로 재빌드 | gitignore. 코드에서 다시 구움 |
| **dist/** | `npm run build` | gitignore |
| **OS/런타임/파이썬패키지/폰트** | `01_작업환경_OS_도구.md` 그대로 | WSL2, node@nvm, Pillow/numpy/rubymarshal, ffmpeg |
| **`.lnk` 바로가기** (`포켓몬 위드 실행.lnk`) | 집 PC에서 exe 가리키게 새로 만들기 | gitignore(각자 빌드한 exe 경로라) |

## 📋 D. 집 PC 복원 최종 체크 (다 됐는지)

```
[ ] git clone / 폴더 동기화 완료
[ ] WSL2 + Ubuntu 24.04, node v22(nvm), python(Pillow/numpy/rubymarshal), ffmpeg 설치
[ ] D드라이브에 어나더레드 원본 폴더 배치
[ ] cd myPokemon_AJ && npm install
[ ] .claude/skills/ 복원 (복사본/claude_skills)
[ ] 메모리 복원 (복사본/claude_memory → 실제 슬러그 폴더)
[ ] .claude/settings.local.json 복원 (선택)
[ ] tools/ar-map/*.py 의 AR/PUB 경로를 집 PC 경로로 수정
[ ] npm run dev → http://localhost:5180 정상
[ ] npm run build && npx electron-builder --win --x64 → Pokemon With.exe 생성
[ ] 이어서: InteriorScene.ts 작성 (03 문서 §5)
```
