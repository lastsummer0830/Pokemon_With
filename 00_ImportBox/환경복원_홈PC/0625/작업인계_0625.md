## 6. 특이사항 / 함정 (꼭 읽기)

- 🔴 **dev 검은 화면 버그**: `/mnt/d`(9P)에서 `usePolling`이 `public/assets`(약 3700개)까지 폴링하면 IO 마비 → 5MB 폰트 서빙이 **34초** 걸려 화면이 검게 멈춤. → `vite.config.ts`의 `watch.ignored`에 `public/assets`, `01_Resources`, `build_win` 등 추가해서 해결함(폰트 34s→0.06s). **이 ignored 지우지 말 것.**
- 🔴 **`npm run app:build` 금지** (리눅스 타겟). 윈도우 exe는 `npx electron-builder --win --x64`.
- ⚠️ **사용자가 막판에 "문제 존나 많다"고 함** — 거실 계단/충돌을 dev로 검증하려다 종료. 즉 **3에서 만든 거실은 코드상으론 맞지만 실제 플레이 검증이 안 끝남.** 집에서 D→4로 들어가 ① 가구 위로 안 올라가지는지 ② 계단 밟고 2층 올라가지는지 ③ 문으로 마을 나가지는지 직접 확인부터.
- ⚠️ **텍스트박스 "실제렌더" 원본 파일을 실수로 덮어썼고 git 미추적이라 복구 불가** — 현재 파일은 재생성본(6종). 사용자에게 사과 완료.
- ⚠️ 주인공 확정: 남=1세대 RED, 여=4세대 DAWN, **픽셀(도트) 전용**(매끈한 일러스트 금지). 라이벌=네모(SV).
